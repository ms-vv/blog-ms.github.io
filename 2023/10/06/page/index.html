分类

反射型

  • 将可以执行的 js 脚本放到 url 参数里面，当用户点击恶意链接时才会执行恶意代码

储存型

  • 恶意代码被保存到数据库中，用户访问网站就会触发恶意 js 代码

  • 这种攻击具有较强的稳定性和持久性

DOM 型

  • 接受数据的过程由 js 实现, 后期代码构造在 html 代码中

  • 与反射型的不同之处在于其不会经过服务端的处理，而是经过 js dom 的渲染构造完 dom 树后，浏览器解析时才会被 js 的解析引擎解析执行。
