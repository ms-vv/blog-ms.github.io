弱口令思路

常见系统弱口令

xxl -job

  • admin: 123456
  • 后台可反弹 shell
  • 默认路径：/xxl-job

K 8 s 控制台

  • admin: P@88w0rd
  • 直接接管集群

Zabbix

  • admin：zabbix
  • 后台可反弹 shell

grafana

  • admin：admin

nacos 系统监控

  • naocs：nacos
  • 未授权，jwt 默认密钥

tomcat

  • admin：admin、tomcat：tomcat
  • 后台可反弹 shell
  • 有专门的字典：tomcat-user|tomcat-pass
  • 数据包中密码被 base 64
    • 选择 base 64
    • 取消 url 编码

ActiveMQ 组件

  • admin：admin

weblogic 控制台

  • weblogic：weblogic

RabbitMQ 组件

  • guest：guest、admin：admin

gitlab 控制台

  • root：可爆破

druid 组件

  • admin：123456

若依

  • admin：admin 123

常见弱口令思路

  • 用户名枚举
    • test-phone 字典测试手机号
      • 固定密码：123456 000000 空密码等
  • 用户登录
    • 固定用户名：admin：对密码爆破
    • 固定密码：123456 对账号进行爆破
      • 推荐用户名字典：常用用户名》中文姓名》top 300》chinaUsername
      • 推荐密码字典：top 19576.txt
    • 可以枚举，先枚举用户再进行密码爆破
    • 国内系统对以下用户进行爆破
      • admin、root、administrator、gly、ceshi、test、student、teacher
  • 密码加密
    • 将简单的 123456 登录之后数据包中复制，爆破常用用户名
    • js 逆向
  • iot 弱口令
    • 推荐字典
      • 常见安全设备默认口令清单
      • 华为安全产品默认用户名密码速查比啊
      • 路由器默认密码
    • cnvd 搜弱口令
      • 大部分都是 admin：123456、admin：admin
